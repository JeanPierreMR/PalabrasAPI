# PalabrasAPI
 
Para hallar frecuencias http://localhost:8080/Frecuencia POST
	Parámetros parrafo (párrafo al cual contar frecuencia de ocurrencia de letras)
	Devuelve las primeras 10
Para verificar palíndromas http://localhost:8080/Palindromas POST
	Parámetros csvList (lista en formato csv simple de palabras)
	Devuelve aquellas que no sean palindromas
